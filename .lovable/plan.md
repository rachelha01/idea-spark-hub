

## Healthcare/Pharmacy Diversification Monitoring Dashboard

A comprehensive analytics dashboard for monitoring diversification metrics in the healthcare and pharmacy industry, featuring a sleek dark mode interface.

### Core Features

**1. Authentication & User Management**
- Secure login/signup system with email authentication
- User profiles with role-based access (admin vs. standard user)
- Password reset functionality

**2. Dashboard Overview Page**
- **KPI Cards** displaying key metrics at a glance:
  - Total revenue across segments
  - Product distribution count
  - Market share percentages
  - Growth indicators with trend arrows
- Quick summary of portfolio health

**3. Interactive Charts & Visualizations**
- **Line charts** for tracking trends over time (revenue, growth rates)
- **Pie/Donut charts** for portfolio distribution breakdown
- **Bar charts** for comparing segments (products, regions, suppliers)
- Charts will be interactive with hover tooltips and click-to-filter

**4. Data Tables**
- Sortable and filterable tables for detailed data exploration
- Product listings, supplier data, regional performance
- Search functionality within tables
- Export capabilities (CSV)

**5. Date Range Filters**
- Global date picker to filter all dashboard data
- Preset options: Today, Last 7 days, Last 30 days, This quarter, This year
- Custom date range selector

**6. Navigation & Layout**
- Collapsible dark-themed sidebar navigation
- Pages: Dashboard, Products, Suppliers, Regions, Reports, Settings
- Responsive design for desktop and tablet

### Design & UX
- **Dark mode theme** throughout with contrasting accent colors
- Clean, professional aesthetic suitable for business use
- Smooth animations and transitions on charts
- Mobile-responsive layout

### Backend (Lovable Cloud)
- Database tables for products, suppliers, regions, metrics
- User authentication and profiles
- Secure data storage with row-level security
